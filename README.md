## Files origin:

Icons are directly extracted from game files with [GCFScape](https://developer.valvesoftware.com/wiki/GCFScape). 

```
steamapps
|-- common
    |-- Counter-Strike Global Offensive
        |-- csgo
            |-- pak01_dir.vpk
```

```
pak01_dir.vpk
|-- root
    |-- materials
        |-- panorama
            |-- images
                |-- map_icons
```
